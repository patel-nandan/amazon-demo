# Amazon Demo

A simple Amazon-inspired e-commerce website built using **HTML, CSS, and JavaScript**. This project demonstrates front-end web development concepts such as DOM manipulation, localStorage, user authentication, product listing, and shopping cart functionality.

## Features

* User Signup
* User Login
* LocalStorage-based Authentication
* Product Listing
* Add to Cart
* View Cart
* Responsive and User-Friendly Interface
* DOM Manipulation with JavaScript
* Form Validation
* Duplicate Email Prevention

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser LocalStorage

## Project Overview

The application allows users to create an account, log in using stored credentials, browse products, and add items to a shopping cart. All user and cart data are stored locally using the browser's LocalStorage, making the project entirely front-end without requiring a backend or database.

## Learning Outcomes

This project demonstrates:

* DOM Manipulation
* Event Handling
* LocalStorage
* Arrays and Objects
* Form Validation
* Dynamic Content Rendering
* Basic E-commerce Workflow
